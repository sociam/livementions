# SOCIAM Livementions Demonstrator
============

Exemplar realtime visualisation of mentions of social machines from the Twitter sample stream.

Demo at:
https://public.indx.ecs.soton.ac.uk/livementions/

# Core Technologies

## Server

* Node.js
* Socket.IO

## Client

* Angular.js
* D3
* Cubism
* Socket.IO
